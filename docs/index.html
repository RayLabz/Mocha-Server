Mocha


A lightweight, multi-threaded TCP/UDP client-server library for Java

Mocha is a lightweight, multi-threaded client-server library written in Java. It enables the quick creation of multi-client, multi-port TCP and UDP clients and servers with minimal overhead. Mocha allows you to quickly set up and run basic servers or define your own custom servers. By using Mocha, you can avoid having to deal with low-level networking technologies like sockets and concentrate on the logic/protocols that are important for your server.
